# Artifacts Directory Policy

This directory is for generated run outputs and temporary benchmark artifacts.

Repository rules:

- Keep only lightweight documentation and placeholder files tracked.
- Do not commit generated files under `code/artifacts/runs/`.
- Do not commit generated split JSON files under `code/artifacts/splits/`.
- Keep `.gitkeep` in `runs/` and `splits/` so directory structure remains stable.

Reproducibility note:

- Canonical run outputs should be regenerated by scripts and recorded in run reports.
- If a result must be preserved for review, export it outside this repository or attach it in release artifacts.
